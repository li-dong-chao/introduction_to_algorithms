#### 题目

（运行时间比较）假设求解问题的算法需要$f(n)$毫秒，对下表中的每个函数$f(n)$和时间$t$，确定可以在时间$t$内求解的问题的最大规模$n$。

|            | 1秒钟 | 1分钟 | 1小时 | 1天  | 1月  | 1年  | 1世纪 |
| ---------- | ----- | ----- | ----- | ---- | ---- | ---- | ----- |
| $\log{n}$  |       |       |       |      |      |      |       |
| $\sqrt{n}$ |       |       |       |      |      |      |       |
| $n$        |       |       |       |      |      |      |       |
| $n\log{n}$ |       |       |       |      |      |      |       |
| $n^2$      |       |       |       |      |      |      |       |
| $n^3$      |       |       |       |      |      |      |       |
| $2^n$      |       |       |       |      |      |      |       |
| $n!$       |       |       |       |      |      |      |       |
